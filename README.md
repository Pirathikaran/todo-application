# TODO App — Full-Stack 

A full-stack TODO application built with React + Vite (frontend) and Node.js + Express + MongoDB (backend).

## Directory Structure

```
Assignments/
├── client/        # React frontend (Vite + Tailwind)
└── server/        # Express + Mongoose backend
```

## Quick Start

### Backend
```bash
cd server
npm install
cp .env.example .env   # then fill in your MONGO_URI
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

Visit `http://localhost:3000` in your browser.

See individual README files inside `client/` and `server/` for full setup details.
